## 보안

Microsoft는 소프트웨어 제품과 서비스의 보안을 중요하게 다루며, 여기에는 GitHub 조직의 모든 소스 코드 저장소가 포함됩니다.

**보안 취약점은 공개 GitHub 이슈로 보고하지 마세요.**

보안 보고 정보, 위치, 연락처, 정책은 Microsoft 저장소에 대한 최신 안내를
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md)에서 확인하세요.
